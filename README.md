# ServletCorsoJava

Per eseguire questa Servlet bisogna scaricare tomcat 8.5 da questo URL https://tomcat.apache.org/download-80.cgi

Una volta avviato tomcat con la webappa deplyata, l'url da raggiungere è:

http://localhost:8080/ServletCorsoJava/comingsoon


Se si vuole vedere la welcome page, bisogna andare su
http://localhost:8080/ServletCorsoJava
